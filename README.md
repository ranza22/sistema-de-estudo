# 🎮 Sistema de Estudos

Projeto feito para praticar HTML, CSS e JavaScript criando um sistema de estudos inspirado em jogos.

## 🚀 Funções

- Sistema de XP
- Level
- Cronômetro
- Metas diárias
- Streak de estudos
- Salvamento com localStorage

## 🛠️ Tecnologias usadas

- HTML
- CSS
- JavaScript

## 💡 Sobre o projeto

Esse projeto foi criado durante meus estudos de programação para treinar lógica, manipulação do DOM e interação com o usuário.

Também utilizei IA como apoio para aprender conceitos, corrigir erros e entender melhor como estruturar algumas partes do código, sempre buscando compreender o funcionamento em vez de apenas copiar.

## 🎯 Objetivo

Meu foco com esse projeto foi praticar programação na prática e evoluir minhas habilidades em desenvolvimento front-end.
